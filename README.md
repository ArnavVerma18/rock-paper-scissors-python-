# rock-paper-scissors-python-
A simple command-line implementation of the famous Rock, Paper, Scissors game built using Python, allowing the user to play against the computer. The computer makes random moves, validates user input, and determines the winner according to the official game rules.

This project was created as part of my Python learning journey to strengthen my understanding of:

- Variables and Data Types
- Dictionaries
- Conditional Statements (`if`, `elif`, `else` )
- User Input Handling
- Random Number Generation
- Error Handling
- Loops
- Formatted Output using f-strings

Features of this game:

-  Random computer choice using Python's `random` module
-  User-friendly input (`r`, `p`, `s`)
-  Accepts both uppercase and lowercase inputs
-  Input validation for invalid entries
-  Automatic win, lose, and draw detection
-  Option to play multiple rounds
-  Beginner-friendly and readable code

Game Logic:

- Rock defeats Scissors
- Scissors defeats Paper
- Paper defeats Rock
- Same choices result in a Draw
